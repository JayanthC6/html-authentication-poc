# html-authentication-poc

CampusPe HTML Authentication System POC.

This assignment contains 5 basic HTML pages that demonstrate a simple authentication flow. As per the requirements, this project strictly uses HTML with no CSS or JavaScript, and all page redirections are handled via anchor tags.
